🏧 ATM Management System
Project Name:
ATM Simulation System — C++ OOP
A console-based ATM simulation built in C++ using Object-Oriented Programming. Designed an ATM class with private balance encapsulation and public methods for secure data access.

Core Features:
- View current balance
- Cash withdrawal with insufficient funds validation
- Cash deposit with updated balance confirmation
- Continuous session loop (repeat transactions until exit)
- Screen clearing for clean user experience

Demonstrates strong command of OOP concepts: Classes, Objects, Inheritance, Encapsulation & Polymorphism.

🛠 Tech: C++ · OOP · Console Application
